# DynamicBungeeAuth (Decompiled & Under Maintenance)

This repository contains the decompiled DynamicBungeeAuth plugin, currently in a buildable state.
The original plugin author appears to no longer maintain it, and the plugin cannot be found on SpigotMC.

## Current Status
- Decompiled and organized into a buildable Java project.
- No significant fixes or optimizations have been applied yet.

## Future Plans
- Optimize this project and remove unnecessary features.
- Gradually transform it into a high-quality JavaPlugin.
- Add support for more Minecraft versions and additional platforms, such as Velocity.

> ⚠️ Note:
> This project is intended for learning, reference, and future maintenance.
> Redistribution or commercial use may violate the original plugin's license.
